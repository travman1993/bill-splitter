## How to Use

### Simple Split Mode
1. Enter total bill amount
2. Set tax percentage (default 8%)
3. Choose tip percentage (15%, 18%, 20% buttons or custom)
4. Enter number of people
5. Click "Calculate"
6. See breakdown and per-person cost

### Itemized Split Mode
1. Add people by name
2. Add items with prices
3. Check boxes to assign items to people
4. Set tax and tip percentages
5. Click "Calculate Splits"
6. See each person's exact share

### Who Pays? Game
1. Add people in Itemized mode first
2. Switch to "Who Pays? Game" tab
3. Click "Spin to Pick Payer!"
4. Random person is selected
5. Click "Spin Again" for new results
